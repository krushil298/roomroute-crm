# Assets

This folder contains essential assets for the RoomRoute CRM application.

## Files

- `image_1762307821152.png` - RoomRoute logo used in:
  - Main application header (`client/src/App.tsx`)
  - Login page (`client/src/pages/login.tsx`)
  - Signup page (`client/src/pages/signup.tsx`)
  - Forgot password page (`client/src/pages/forgot-password.tsx`)
  - Reset password page (`client/src/pages/reset-password.tsx`)
  - Landing page (`client/src/pages/landing.tsx`)

## Important

**Do not delete this folder or the logo file** - it is required for the application to build and run properly.

The logo is imported as `@assets/image_1762307821152.png` throughout the application.
